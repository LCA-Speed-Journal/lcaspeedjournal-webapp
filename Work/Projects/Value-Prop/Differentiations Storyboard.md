## Slide 8: Preconstruction - Value Engineering by Work Packaging

**Slide Title:** Value Engineering by Work Packaging

**Content:**
*Narration:* "Our approach: Plan by Work Package, Schedule by Work Package, Track by Work Package. We utilize Advanced Work Packaging, a construction-driven planning system focused on creating a constraint-free work environment. This includes Construction Work Packages at the highest level, Installation Work Packages at the smallest level, and Workforce Planning to break down CWPs into discrete execution packages."

*On-screen text (progressive reveal with definitions):*
**Process Flow:**
Plan by Work Package → Schedule by Work Package → Track by Work Package

**Advanced Work Packaging (AWP):**
Construction-driven planning and collaboration system for building capital projects, sharply focused on creating a constraint-free work environment.

**Construction Work Packages (CWP):**
Highest level work package. Identified by type of work. Examples: Overhead Conduit, Cable Tray, Grounding, etc.

**Installation Work Packages (IWP):**
Smallest type of work package on a project. Contains scope of work that allows specific crew to operate independently for a specified period.

**Workforce Planning (WFP):**
The process of breaking down high-level CWPs into discrete IWPs for execution in the field. Typically known as workflow.

*Interactive element:* Hierarchical diagram showing CWP → IWP breakdown (click to expand levels)

**Design Ideas:**
- Visual hierarchy diagram: AWP → CWP → IWP
- Click to expand each level and see examples
- Process flow arrows showing Plan → Schedule → Track
- Icons for each work package type
- Example work packages shown (Overhead Conduit, Cable Tray, etc.)
- Interactive tree diagram
- Color coding by package level
- Expandable examples for each package type
