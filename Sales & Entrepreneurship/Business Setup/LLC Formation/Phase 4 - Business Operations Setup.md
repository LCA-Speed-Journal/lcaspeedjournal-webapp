# Phase 4: Business Operations Setup

## Online Presence and Marketplace
- [ ] Set up business website (if not using Inc Authority add-on)
- [ ] Create accounts on online marketplaces (e.g., Trainerize, TrueCoach, or custom platform)
- [ ] Set up payment processing (Stripe, PayPal Business, Square)
- [ ] Configure sales tax collection (if applicable)
- [ ] Create terms of service and privacy policy for online services

## Accounting and Bookkeeping
- [ ] Choose accounting software (QuickBooks Online, Xero, or FreshBooks)
- [ ] Set up chart of accounts
- [ ] Connect business bank account
- [ ] Set up invoicing system
- [ ] Create expense tracking system (receipts, mileage, etc.)

## High School Contractor Setup
- [ ] Verify NHSSCA certification is current
- [ ] Draft contractor agreement with school district
- [ ] Include: scope of work, compensation, liability, insurance requirements, term
- [ ] Review with school district legal/business office
- [ ] Obtain required background checks (if needed by district)
- [ ] Provide certificate of insurance to school district

## Diagnostic Services Setup
- [ ] Determine pricing for sprint-profiling and force-dynamometry services
- [ ] Create service agreements/contracts for team clients
- [ ] Set up scheduling system
- [ ] Prepare equipment and protocols
- [ ] Create reporting templates for diagnostic results

## Programming Marketplace Setup
- [ ] Create programming templates/products
- [ ] Set up digital delivery system
- [ ] Configure pricing and payment options
- [ ] Create product descriptions and marketing materials
- [ ] Set up customer support system
