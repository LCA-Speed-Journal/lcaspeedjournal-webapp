# Overview
Split: **6x p/wk** — 2x Upper, 2x Lower, 2x “Field-Days”
- **Accel**: Contrast-Sprints
- **Squat**: Heel-Elevated SQ Focus (plus Contrast Work)
- **Upper-Push**: BB Bench-Focused
- **MaxV**: 30-40yd Sprints
- **Hinge**: Olys, RDLs
- **Upper-Pull**: Chin-Focused

# Block 2: Weeks 6-10
## Accel-Days: 
- Repeat the sets progression but with 25yd unresisted
- Hurdle Hops are now Medium-Hops (2x10, progress as [[Daniel Bach]])
## Squat-Days:
- Heel-Elevated SQ now a 3x3
- SQ Superset now a Tri-Set: Heel-Elevated → DB Jump SQ → Depth-Jump to Box (~20”)
- Reframed Accessories: 3 Sets of
	- Leg Curl (x10) → Seated Calf (x10) → MB Lunge Jumps (x20)
- New Prehab/Rehab Superset:
	- Ab Rollout (3 RIR) → fRoller Bridge w/OSC (30s)
## Upper-Push
- Bench now a 3x3
- Add weight to deficit push-ups
## MaxV
- Reset to 6 sets, but 35yds
- Swap Medium Hops for SL Hops: 3 x 5
## Hinge-Days
- Swap Hurdle-Hops for Drop-Jumps: 3x5
- Olys are 3x2 now
- New Main-Superset: CON-Only Trap-Bar DL (3x3 @ 3 RIR) → MB Lateral-to-Vertical Hops (3x3)
- New Secondary: BB RDL (3x8) → Bulgarian Split-SQ (8x p/ea) → Standing Calf
- Swap Standing-Calf in Accessory: GHD Russian-Twist
## Upper-Pull
- Weighted-Chin now 3x3
- Row Superset Swap: DB Incline (x10) → Wrapping Lat-Pull (x10 p/ea)
- Tri-Set: Wide-Grip Row → Dips → High-Fly (Cable)
