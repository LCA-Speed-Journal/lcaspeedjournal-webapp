# Overview
- 4 Days p/wk (Upper-Lower || Rest || Upper-Total)
- 3 Phases: 8x5 → 10x4 → 12x3
	- Clusters used on Primary-Lifts, accessories are kept more to basics
- Test 3RM or 1RM on Week-1 of Each Phase
- **Clusters on Main-Lifts:** Bench-Row; Press-Pull; Squat; Deadlift
	- Upper (supersets): 70-75% → 75-80% → 80-85%
	- Lower (straight-sets): 65-70% → 70-75% → 75-80%
- Arms/Shoulders are lower-weight, higher-volume, twice-weekly p/ea
	- 4x25 / 3x30

# Breakdown
## Phase 1: 8x5 (3wks)
### Upper: Bench/Row || Push-Press/Pull-Up
- Warmup: MB Throws
- **Main-Lift: Push-Pull Superset** — 8x5 / 8x10
	- 30-45s Rest between each
- (Day-1 Upper) Secondaries
	- DB Shoulder Press: 3x10
	- BW Push-Up: AMRAP
- (Day-2 Upper) Arms + Shoulders
	- Bicep-Tricep Superset A: 3x15 p/ea
	- Bicep-Tricep Superset B: 3x15 p/ea
	- Shoulders-Superset: 4x25 p/ea
### Lower: Front-Squat
- Warmup: trap-Bar Jumps
- Front-Squat
- DB Step-Ups
- BB Glute Bridge
- BW Back-Extension
### Total: DL + Upper-Capacity
- Warmup: Resisted Sprints
- Deadlift
- Push-Pull Superset (DB Bench / DB Row)
- Forward-to-Reverse Lunge
